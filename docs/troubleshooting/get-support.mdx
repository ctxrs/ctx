Report issues at:

- https://github.com/ctxrs/ctx/issues
- support@ctx.rs for support requests that should not live in a public issue

## Legal and trust pages

- [Privacy Policy](/legal/privacy-policy)
- [Terms of Use](/legal/terms)
- [Security](/legal/security)

## Include this minimum info

- what you tried to do
- what happened
- what you expected
- exact error text
- OS and app version

## Helpful extras

- task/session context (without sensitive code)
- screenshots of relevant UI state
- whether issue reproduces after app restart
