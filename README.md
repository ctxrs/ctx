# ctx

Public issue tracker for **ctx**.

ctx is an agent workbench for real codebases. It lets you run coding-agent tasks from a desktop GUI, review diffs, and apply changes safely.

## Install

```bash
curl -fsSl https://ctx.rs/install | sh
```

## Docs

- https://ctx.rs/docs

## Scope of this repository

- Use this repository for issue reports and discussions about ctx.
- Product source code is maintained in internal repositories.
