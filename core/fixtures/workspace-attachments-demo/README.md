# Workspace Attachments Demo

This folder is a template for testing workspace attachments.

Usage:
1) Copy this folder to a scratch location (e.g. `/tmp/ctx-attachments-demo`).
2) Run `git init`, add a dummy file, and commit once.
3) Register the copied folder as a workspace in ctx.
4) Sync attachments to materialize the React repo and docs.

Attachments configured in Settings are stored in local daemon workspace state.
