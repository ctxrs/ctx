# Contributing

Thanks for improving ctx.

This repository is the public GitHub home for ctx overview content, selected documentation, README media, and issue tracking. The ctx product itself is closed source.

## What belongs here

- `docs/`: markdown or MDX documentation intended for GitHub browsing
- `assets/`: screenshots, logos, README media, and demo artifacts
- `.github/`: issue templates and repository configuration

## Working style

- Prefer small, reviewable changes.
- Keep the top-level structure clean and easy to browse on GitHub.
- Prefer repo-readable docs over site build plumbing.
- If a docs page works better on the rendered site, keep the GitHub version readable and link to `https://ctx.rs` when useful.

## Validation

- Preview `README.md` on GitHub after media changes.
- Check new docs for obvious broken links.
- Keep large binary assets intentional and documented.
